# event_kampus_locator

Event Kampus Locator
Aplikasi Flutter sederhana untuk mendeteksi lokasi pengguna, melakukan tracking posisi, menghitung jarak ke event kampus, serta menampilkan peta menggunakan OpenStreetMap (OSM).

Fitur Utama
• Mengambil lokasi saat ini (Get Current Location)
• Tracking lokasi secara realtime (Start / Stop Tracking)
• Reverse Geocoding (alamat berdasarkan koordinat)
• Daftar event kampus + jarak dari posisi user
• Navigasi ke halaman peta (OSM Map)
• Pengaturan akurasi dan distance filter
